
🌍 Arabic Gender Lookup CLI
Powered by Groq & Qwen 3 32B

An automated tool for Arabic language learners to instantly identify the grammatical gender of any object. Instead of manual dictionary searches, this tool leverages Qwen 3 32B to provide the Arabic translation, Harakat (vowels), and the specific grammatical rule for the gender.

🚀 Features
Instant Inference: Powered by Groq’s LPU for sub-second responses.

Linguistic Reasoning: Uses Qwen 3 to explain why a word is masculine or feminine (detecting exceptions like "Sun" or "Wind").

Vocalized Output: Includes Harakat for correct pronunciation.

Simple CLI: A clean, infinite-loop command line interface for rapid-fire lookups.

🛠️Setup - 
 1. Go to https://groq.com
 2. Click on Start Building.
 3. Then  Signin or Login.
 4. After signing up click on api key.
 5. Create a new a key named qwen/qwen3-32b (Keep that api key safely and copy and paste it in notepad first).
 6. Then copy this code.
 7. After copying paste it in visual studio code.
 8. Paste the api key in line number 6.
 9. Congratulations now you have your own arabic object gender finder .
